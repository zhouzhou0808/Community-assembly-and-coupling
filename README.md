Project Overview：        
    This repository contains all R analytical scripts for the manuscript currently under submission, focusing on bacterial community assembly, multi-media microbial interactions, and the ecological effects of wintering waterbirds in lake wetland ecosystems. 

Data Instructions：        
    Raw sequencing data and environmental metadata are not included in this code repository to protect unpublished data of the submitted manuscript.
    
Input data format requirements:        
    Species abundance matrix: rows = taxa (phylum/genus), columns = samples
    Physicochemical table: sample ID + water/sediment indicators (nutrients, chlorophyll a, etc.)
    Grouping information: early wintering period / late wintering period, water/sediment media
    
Analysis Workflow Suggestion：        
    Preprocess sequencing data → Calculate relative abundance
    Basic community profiling: Other analysis.R (species composition & inter-group differences)
    Environmental gradient ordination: PCA.R (physicochemical PCA)
    Community assembly process: NCM.R (neutral model fitting)
    Microbial co-occurrence network: WGCNA.R → Number of modules.R (network & module analysis)
    Multi-medium driving mechanism: PLS-PM.R (path analysis)
    
Important Notes：        
    Manuscript status: The corresponding paper is currently under journal review. Please do not share or reuse these codes without prior consent of the author.
    Parameter adjustment: Sample names, grouping labels, file paths and filtering thresholds need to be modified according to your own dataset before running.
    Outputs: All figures, statistical tables and model parameters generated by these scripts can be directly used for manuscript figures and supplementary materials.
    Contact: For code errors or analytical questions, please contact the corresponding author.
